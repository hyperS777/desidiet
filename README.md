# desidiet 
Indian meal diet tracker with AI-based nutrition estimation

Problem Statement:
## Problem Statement
Most diet tracking apps focus on Western food and make it hard to track common Indian meals like roti, dal, dosa, and biryani. Users either guess nutrition values or stop tracking altogether.

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (FastAPI)  
- AI: OpenAI API  

## Features
- Add meals from Indian food list  
- Enter nutrition manually  
- Estimate nutrition using AI  
- View daily calorie and macro summary  

## Setup Instructions
1. Clone the repository  
2. Open `frontend/index.html` in browser  
3. Run backend using:
   ```bash
